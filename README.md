# Grocery Store POS Ecosystem

Enterprise-grade Point of Sale system for modern supermarkets with complete Windows desktop, Android mobile, and backend API integration.

## 🏗️ Project Structure

- **backend-api/** - ASP.NET Core REST API + SignalR
- **windows-pos-app/** - .NET WPF Desktop Application
- **android-app/** - Kotlin/Jetpack Compose Mobile App
- **database-schema/** - PostgreSQL Database Design
- **docs/** - Documentation & Guides

## 🎯 Core Features

✅ Real-time POS Billing
✅ Barcode Scanning & Inventory
✅ Multi-user with Role-Based Access
✅ Real-time Synchronization (WebSocket/SignalR)
✅ Offline-First Architecture
✅ Receipt & Label Printing
✅ Delivery Management
✅ Customer Loyalty System
✅ Analytics & Reporting
✅ Arabic + English Multi-language Support

## 🛠️ Technology Stack

**Backend**: ASP.NET Core, Entity Framework Core, PostgreSQL, SignalR
**Windows**: .NET WPF, MVVM, Prism
**Android**: Kotlin, Jetpack Compose, Room, Retrofit, WorkManager
**Real-time**: WebSocket, SignalR, Firebase Cloud Messaging

## 📋 Branches

- `main` - Production ready code
- `backend-api` - Backend development
- `windows-pos-app` - Windows desktop app
- `android-app` - Android mobile app
- `database-schema` - Database & migrations
- `deployment` - Docker & deployment configs
